Once the app opens, you will see a simple window.

- **Load Night Light Data:** The software automatically loads VIIRS night lights data for different dates.  
- **Change Detection:** Compare different time points to see where lights have dimmed or brightened.  
- **Map View:** A map shows you the location of changes clearly.  
- **Export Results:** Save images or reports for later use.

### Basic Steps to Get Started

1. **Select Dates:** Use the date picker to choose two or more time points.

3. **View Results:** Look at colored areas on the map showing increases or decreases in light.  
4. **Save Your Work:** Use the "Export" option to save your map or data summary.
